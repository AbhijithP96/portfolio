Computer Vision Intern, Carl Zeiss Meditec AG

During my internship at Carl Zeiss, I developed a custom Python-based GUI toolbox for camera calibration, providing an intuitive interface to calibrate cameras using statistical methods and standard patterns (checkerboard, circular grid, Charuco board). The toolbox integrated OpenCV algorithms and custom MATLAB functions via the MATLAB-Python API, and included robust outlier handling for Charuco detection. Key features included real-time calibration feedback, visualization of detected/projected features, distortion maps, and calibration parameters. The tool also supported stereo calibration and provided detailed visualizations such as per-image reprojection errors and pixel-wise feature errors. The project followed clean code practices with thorough documentation. Additionally, I worked on a depth estimation model to generate pseudo-right images for 3D visualization, enabling stereoscopic viewing with 3D glasses.
